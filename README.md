# Numpy Neural Network

A from-scratch neural network implementation built entirely with NumPy.

## 🚧 Status
This project is currently in the early stages of development. The main objective is to implement forward and backward propagation without relying on high-level deep learning frameworks like PyTorch or TensorFlow, purely using vectorization and linear algebra with NumPy.

## 🚀 Future Features
- Full forward propagation and backward propagation implementation.
- Various activation functions (ReLU, Sigmoid, Softmax).
- Customizable layers and architecture.

## 📦 Requirements
- `numpy`
